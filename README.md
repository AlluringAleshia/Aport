Welcome to my personal portfolio repository! This website serves as a central hub to showcase my background, highlight academic and development projects, and share my journey as a software developer.

##  Features

- **Responsive Hero Section:** Features a clean greeting, professional summary, and call-to-action buttons.
- **Interactive Elements:** Features a responsive layout with smooth hover transition animations built into the profile image.
- **Social Integration:** Fast, direct links to professional networks and contact channels.
- **Clean Styling:** Handcrafted custom design utilizing modern CSS layout principles.

## Tech Stack

- **HTML5:** Structured semantic markup.
- **CSS3:** Custom layouts, interactive animations, and responsive styling.
- **Google Fonts:** Clean modern typography tracking.
